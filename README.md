# Network Optimization
BBR+TCP 网络优化一键脚本

## 一键优化
```
bash <(curl -sL https://raw.githubusercontent.com/yahuisme/network-optimization/main/script.sh)
```

## 脚本特色
✅ 智能检测: 自动识别内存、CPU、虚拟化环境

✅ 精准分层: 6个清晰的内存划分区间

✅ 动态优化: 参数根据硬件规格自动调整

✅ 即时生效: 无需重启，立即享受性能提升

✅ 安全可靠: 自动备份、错误检查、可回滚

✅ 标准路径: 使用 /etc/sysctl.d/99-bbr.conf
